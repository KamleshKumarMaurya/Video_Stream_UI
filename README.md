# Video Streaming UI 🎬

Frontend application for the **Video Streaming Platform**.
This project is built using **Ionic Angular** and communicates with a **Spring Boot backend** to stream video content.

---

## 🚀 Features

* Video streaming interface
* Responsive mobile-friendly UI
* Video player integration
* List of available videos
* REST API integration with backend
* Ionic mobile UI components

---

## 🛠 Tech Stack

* Ionic Angular
* Angular
* TypeScript
* HTML5
* CSS3

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/KamleshKumarMaurya/Video_Stream_UI.git
```

Go to project folder:

```
cd video_streaming
```

Install dependencies:

```
npm install
```

Run the project:

```
ionic serve
```

The application will start at:

```
http://localhost:8100
```
